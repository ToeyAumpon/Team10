
# TEAM10

## Introduction

A simple community app built by IT students (TEAM10). A digital platform designed to connect individuals by sharing messages and photos with features like login, tagging, and search functionality.

## Concept

This application enables users to:
- Create and share posts with text and photos
- Connect with other community members
- Discover content through tags and search
- Securely manage their accounts with login authentication

**Backend:**
- Java

**Architecture:**
- RESTful API
- Client-Server Architecture

## How to Use

1. **Register/Login**: Create an account or login with existing credentials
2. **Create a Post**: Click "New Post" to share text and/or images
3. **Browse Posts**: View posts from other community members on the main feed
4. **Search & Filter**: Use the search bar and tags to find specific content
5. **View Details**: Click on a post to see full details and comments
6. **Manage Account**: Update profile settings and manage your posts
