# TEAM10

## はじめに

IT学生によって開発されたシンプルなコミュニティアプリ（TEAM10）。ログイン、タグ付け、検索機能などの機能を備えた、メッセージと写真の共有を通じて個人を繋ぐデジタルプラットフォームです。

## コンセプト

このアプリケーションは以下の機能を提供します：
- テキストと写真を含むポストの作成と共有
- コミュニティメンバーとの繋がり
- タグと検索によるコンテンツの発見
- ログイン認証による安全なアカウント管理

## 技術スタック

**フロントエンド：**
- HTML5
- CSS3
- JavaScript

**バックエンド：**
- Java
- Spring Bootフレームワーク

**アーキテクチャ：**
- RESTful API
- クライアント・サーバアーキテクチャ

## 使い方

1. **登録/ログイン**: アカウントを作成するか、既存の認証情報でログイン
2. **ポストを作成**: 「新しいポスト」をクリックしてテキストと/または画像を共有
3. **ポストを閲覧**: メインフィードで他のコミュニティメンバーのポストを表示
4. **検索とフィルター**: 検索バーとタグを使用して特定のコンテンツを検索
5. **詳細を表示**: ポストをクリックして完全な詳細とコメントを表示
6. **アカウント管理**: プロファイル設定を更新し、ポストを管理

## 計画されている機能

- ユーザープロファイルのカスタマイズ
- ユーザー間のダイレクトメッセージング
- ポストのいいねとリアクション
- ネストされた返信を含むコメントシステム

## 終わりに

このプロジェクトはTEAM10のフロントエンドとバックエンド開発での協力能力を示し、
コミュニティの相互作用とエンゲージメントのための機能的なプラットフォームを作成します。
